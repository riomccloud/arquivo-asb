Porte de arenas Batalha do SRB2Kart versão 1.x para o SRB2 Battle é um add-on incompleto de pacote de mapas para o Sonic Robo Blast 2.

Descrição

Porte experimental das arenas de modo de jogo Batalha do Sonic Robo Blast 2 Kart versão 1.2 para o modo de jogo SRB2 Battle do Sonic Robo Blast 2.
