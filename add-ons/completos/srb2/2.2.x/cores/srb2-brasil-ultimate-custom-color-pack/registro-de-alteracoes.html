Registro de alterações do add-on "SRB2 Brasil Ultimate Custom Color Pack"

Versão 2.5

  • A correção de animações agora é incluída no próprio add-on;
  • Foram adicionadas 11 cores novas.

Versão 2.1

  • Adicionado suporte oficial a versão 2.2.7 Release Candidate 1 do Sonic Robo Blast 2;
  • Foram removidos espaçamentos nos nomes de algumas cores;
  • O nome da cor “Aqua” foi alterado para "Water", para que não houvesse conflito com a cor honônima do jogo base.

Versão 2.0

  • Foi adicionada uma correção de animações opcional em um arquivo separado;
  • Foram adicionadas 130 cores novas;
  • Foram corrigidos problemas menores;
  • O nome do add-on agora é “SRB2 Brasil Ultimate Custom Color Pack”.

Versão 1.0

  • Lançamento inicial.
