F. Sonic é um add-on de personagem para o Sonic Robo Blast 2.

Descrição

O mesmo personagem do jogo base, mas com visual renovado se comparado ao presente no jogo, além de possuir pequenas mudanças de jogabilidade.

Habilidades

Nome Tipo Descrição
Aceleração Máxima Passiva Permite o jogador correr mais rápido após acumular momento linear.
Spin Dash Ativa A mesma que a de Sonic.
