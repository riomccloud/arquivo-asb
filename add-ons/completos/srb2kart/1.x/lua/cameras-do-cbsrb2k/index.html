Câmeras do CBSRB2K é um add-on Lua para o Sonic Robo Blast 2 Kart.

Descrição

Adiciona diversas câmeras pelo percurso das 60 pistas do jogo base, semelhante as utilizadas na Fórmula 1. Para utilizá-las, entre no modo Espectador, abra o console do jogo, digite o comando mudarcamera e pressione o botão CUSTOM3 para ativar a visualização das câmeras. Utilize as teclas CUSTOM1 e CUSTOM2 para retroceder e avançar pelas câmeras.

Para criar uma câmera, carregue o add-on de criação sozinho e entre na pista desejada. Utilize os botões FORWARD, BACKWARD, TURNLEFT e TURNRIGHT para navegar pelo mapa. Utilize a combinação DRIFT + FORWARD ou BACKWARD para mudar o ângulo da câmera. Ao estacionar no ponto desejado, aperte o botão CUSTOM3 para marcar o ponto. Após marcar todos os pontos, digite o comando imprimirtabela no console para imprimir as coordenadas no arquivo de log. Após isto, abra-o, copie o trecho com as coordenadas e aplique-as no add-on da forma que desejar.

Os autores recomendam o uso do add-on NameTags Edição CBSRB2K em conjunto para melhor visualização.
